# 新启航英语学习系统 - 功能清单

## 产品概述

小学生英语单词学习系统（1-3年级），支持 Web 浏览器和 Android APP 跨平台使用。

## 核心功能

### 1. 首页仪表盘
- 显示统计数据：单元数量、单词总数、待复习错题数
- 快速入口：开始学习、错题集、成就、打卡、单词管理、设置

### 2. 学习模式

#### 2.1 单词学习模式
- 显示单词卡片（单词、音标、中文意思）
- 标准发音播放（点击喇叭按钮或自动播放）
- **跟读评分功能**：
  - 点击"我来跟读"开始5秒倒计时录音
  - 语音识别用户发音
  - 基于编辑距离计算相似度，评分 0-5 星
  - 语音反馈（Perfect/Good/Try again）
  - 支持手动结束录音
- 切换到下一个单词
- 完成学习模块时自动记录打卡

#### 2.2 单词认识模式（测验）
- 自动播放单词发音
- 显示英语单词和音标
- 4个中文选项供选择
- 答对/答错动画反馈
- 答错自动收录到错题集
- 完成测验模块时自动记录打卡

#### 2.3 单词拼写模式（测验）
- 点击喇叭播放发音
- 选择字母组成单词
- 支持字母重置、提示功能
- 答错自动收录到错题集
- 完成测验模块时自动记录打卡

### 3. 错题集

- 查看所有错题（支持按类型筛选：全部/单词认识/单词拼写）
- 点击错题进入复习界面
- 复习时重新作答
- **掌握判定**：连续答对2次自动从错题集移除
- 清空所有错题功能

### 4. 成就系统

#### 4.1 成就徽章
- 16种成就徽章分类：新手、连续、学习、复习、时间
- 自动检测成就解锁条件
- 首次解锁时全屏弹窗展示
- 成就展示页面（查看已获/未获徽章）
- 进度追踪（未获徽章显示获取进度）

### 5. 打卡系统

- 日历视图展示打卡记录
- 绿色标记表示已打卡日期
- 月份/年份切换导航
- 今日快速定位按钮
- 本月打卡统计（打卡天数、完成率）
- 打卡规则说明

**打卡规则**
- 每天完成"单词学习"、"单词认识"或"单词拼写"中任意一个模块即可打卡

### 6. 单词管理

#### 单元管理
- 查看所有单元列表
- 添加新单元
- 编辑单元名称
- 删除单元（同时删除单元下所有单词及关联错题）

#### 单词管理
- 查看指定单元的单词列表
- 添加单词（单词、音标、中文意思）
- 编辑单词信息
- 删除单词

#### 批量导入
- 支持 CSV/TXT 格式导入
- 格式：`单词,中文意思,音标`
- 预览功能，确认后批量添加

### 7. 设置

#### 数据管理
- **导出数据**：JSON 格式备份
  - Web：弹出文件保存对话框
  - APP：直接下载到系统下载目录
  - 包含单元、单词、错题、设置、成就、打卡记录、自然拼读进度
- **导入数据**：选择 JSON 文件恢复数据
- 导入时显示各类数据统计预览

#### 学习设置
- 音效反馈开关
- 每次练习题目数（5/10/15/20/30题）
- 每日学习目标（5/10/20/30题）

### 8. 数据持久化

- 使用 Zustand + localStorage 存储
- 数据包括：单元、单词、错题记录、成就、用户设置、打卡记录、自然拼读进度
- 页面刷新/重启数据不丢失
- 本地视频使用 IndexedDB 持久化存储（base64 编码）

### 9. 课堂教学视频

#### 9.1 内置视频
- 预置11个课堂教学视频（Unit 1-5 + 字母拼读系列）
- 视频网格展示，点击直接播放

#### 9.2 本地视频上传
- 支持从本地选择视频文件上传
- 视频数据持久化存储到 IndexedDB
- 上传后立即可播放，刷新不丢失
- 支持删除本地视频

#### 9.3 视频播放
- 全屏视频播放，自动播放
- 标准播放控制条

### 10. 自然拼读

#### 10.1 字母选择
- 26字母网格展示（4列布局）
- 已学习字母显示为绿色
- 未学习字母显示为白色

#### 10.2 视频播放
- 点击字母播放对应视频
- 学习完成后自动标记已学
- 可重新观看已学习字母的视频

#### 10.3 学习进度
- 26字母学习进度追踪
- 导出/导入时包含学习进度

## 技术特性

### 跨平台支持
- Web 应用（Vite + React + TypeScript）
- Android APP（Capacitor 打包）

### 语音功能
- 文字转语音（TTS）：Web Speech API + Capacitor TTS 插件
- 语音识别：Web Speech API + Capacitor Speech Recognition 插件

### 响应式设计
- 适配手机屏幕
- 大按钮、高对比度配色，适合小学生使用

## 数据模型

```
Unit
├── id: string
├── name: string
├── words: Word[]
└── createdAt: string

Word
├── id: string
├── unitId: string
├── word: string
├── phonetic: string
└── meaning: string

Mistake
├── id: string
├── wordId: string
├── word: string
├── meaning: string
├── correctAnswer: string
├── type: 'recognition' | 'spelling'
├── options?: string[]  // 认识模式选项
├── wrongAnswer: string
├── times: number       // 错误次数
└── mastered: boolean   // 是否已掌握

CheckInRecord
├── date: string       // YYYY-MM-DD 格式
├── completedModules: ('learning' | 'recognition' | 'spelling')[]
└── timestamp: number

LocalVideo
├── id: string
├── name: string
├── path: string        // base64 编码的视频数据
├── size: number        // 文件大小（字节）
└── addedAt: number     // 上传时间戳
```
