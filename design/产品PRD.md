## 1. 产品概述

### 1.1 产品背景

小学英语学习市场规模持续增长，家长对孩子的英语学习投入持续增加。现有的英语学习APP多面向成人或高年级学生，针对1-3年级（6-9岁）小学生的产品较少，且普遍存在以下问题：

- 学习方式枯燥，孩子容易厌倦
- 缺乏即时激励，孩子坚持不下来
- 家长无法了解孩子学习情况
- 复习机制缺失，导致遗忘率高

### 1.2 产品定位

**目标用户**：小学1-3年级学生（6-9岁）及学生家长

**核心价值主张**：用游戏化方式让小学生爱上英语学习，用智能复习机制确保学习效果

**差异化优势**：
- 游戏化积分成就系统，持续激励学习
- 艾宾浩斯遗忘曲线驱动的智能复习
- 家长端学习报告，了解孩子学习情况
- 针对低年级学生的友好界面设计

### 1.3 目标

| 指标 | 目标值 |
|------|--------|
| 日活用户留存（7日） | ≥40% |
| 单次学习时长 | ≥5分钟 |
| 错题掌握率（复习后） | ≥80% |
| 用户续费率（付费功能） | ≥60% |

### 2.2 学习模式

#### 2.2.1 单词学习模式 ✅

**已完成功能**
- 单词卡片展示（单词、音标、中文意思）
- 标准发音播放（点击喇叭或自动播放）
- **跟读评分功能**：
  - 5秒倒计时录音
  - 语音识别（Web Speech API / Capacitor Speech Recognition）
  - 编辑距离算法计算相似度
  - 0-5星评分展示
  - 语音反馈（Perfect/Good/Try again）
  - 手动结束录音支持

**评分算法**
```
similarity = (单词长度 - 编辑距离) / 单词长度
score = max(1, round(similarity * 5))
```

**交互流程**
1. 点击"我来跟读" → 播放标准发音 → 开始5秒倒计时
2. 用户跟读 → 语音识别 → 计算相似度 → 显示评分
3. 显示星星评分和鼓励文字 → 进入下一单词

### 2.3 错题集 ✅

**已完成功能**
- 查看所有未掌握错题
- 按类型筛选（全部/单词认识/单词拼写）
- 点击错题进入复习界面
- 复习时重新作答
- **掌握判定**：连续答对2次自动从错题集移除
- 清空所有错题功能

**数据结构**
```typescript
interface Mistake {
  id: string;
  wordId: string;
  word: string;           // 单词
  meaning: string;       // 中文意思
  correctAnswer: string;  // 正确答案
  type: 'recognition' | 'spelling';
  options?: string[];     // 认识模式选项
  wrongAnswer: string;   // 错误答案
  times: number;         // 错误次数
  mastered: boolean;     // 是否已掌握
}
```

**复习流程**
1. 选择错题 → 进入答题界面
2. 重新作答 → 判定正确/错误
3. 答对 → times+1，连续2次答对 → mastered=true
4. 答题后1.5秒自动返回列表

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | <2秒（3G网络） |
| 操作响应时间 | <100ms |
| 语音识别延迟 | <1秒 |
| APK大小 | <30MB |

### 5.2 兼容性

- Android：5.0+
- iOS：12.0+
- 浏览器：Chrome/Safari最新版

### 5.3 数据安全

- 用户数据本地加密存储
- 传输使用HTTPS
- 敏感操作需二次确认

### 5.4 辅助功能

- 深色模式支持
- 字体大小调节
- 语音播报开关
